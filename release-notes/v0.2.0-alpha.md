# Remasters RPG v0.2.0-alpha

## Highlights

- New cast spell system for abilities with a dedicated cast chat card.
- Ability item gained an **Additional Settings** tab for attack/secondary/damage overrides.
- Automatic Reiki checks now trigger on ability cast based on casting options.
- Major roll chat improvements:
  - Unified style for check, damage, reiki, and cast cards.
  - Better outcome display and GM-only target info handling.
  - Expanded damage application flow (damage/half/double/soak).
- Default icon for new ability items:
  - `systems/remasters-rpg/assets/icons/spiky-explosion.png`

## Quality and UX

- Better item-sheet scroll persistence during updates.
- Improved cast button reliability with robust actor/item source resolution.
- Multiple ability, inventory, and action tab interaction refinements.
- Expanded EN/PT-BR localization coverage.
